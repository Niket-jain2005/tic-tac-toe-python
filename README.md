# Tic-Tac-Toe Game

A console-based Tic-Tac-Toe game developed in Python as part of the InternPe Python Programming Internship (Task 2).

## Features

* Player vs Computer gameplay
* Input validation
* Win detection (Rows, Columns, and Diagonals)
* Tie detection
* Random computer moves

## Technologies Used

* Python
* Random Module

## How to Run

```bash
python tictactoe.py
```

## Project Overview

This project allows a user to play Tic-Tac-Toe against the computer in the terminal. The game checks for winning conditions, handles ties, validates user input, and generates random moves for the computer.

## Learning Outcomes

* Python Functions
* Loops and Conditional Statements
* Lists and Indexing
* Problem Solving
* Game Logic Implementation
